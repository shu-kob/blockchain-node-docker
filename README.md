# blockchain-node-docker

## How to Use

```
$ git clone https://github.com/shu-kob/blockchain-node-docker
cd blockchain-node-docker
```

### For example

#### How to build bitcoin Docker Image

```
$ cd bitcoin
$ ./build.sh v25.0
```

##### Run Bitcoind Docker

```
$ docker-compose up
```

#### How to build bitcoin cash node Docker Image
